1. You can now upload a json file with this collection format:

[{
  "@shape" : "Hexagon",
  "side" : 3.0
},
{
  "@shape" : "TriangleEquilateral",
  "side" : 3.0
},
{
  "@shape" : "Square",
  "side" : 3.0
},
{
  "@shape" : "Pentagon",
  "side" : 3.0
}]

and you will get back as a response a json with your objects
along with the perimeter and area calculated.
